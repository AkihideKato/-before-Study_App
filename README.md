直感的に分かりやすいようにデザインしました。
いいね機能を「復習」「再復習」とする事で学習の不足部分を分かりやすいようにしました。

## 導入

$ git clone https://github.com/AkihideKato/Study_App  
$ cd Study_App  
$ bundle install  
$ rails db:create  
$ rails db:migrate  
$ rails s  
👉 http://localhost:3000
